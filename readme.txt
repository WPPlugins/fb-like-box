== Description ==


FB Like Box will help one to insert the blue rounded rectangular facebook like box below every post .It even includes a facebook fan button


== Installation ==

1. Upload the .zip file at wp-content/plugins and extract it

2. Activate it 

3. Fill the form at settings/FB like After Post Content
